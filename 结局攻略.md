# 《列城之主》结局攻略

> 本文包含完整结局触发条件，建议至少游玩几局后再看。

## 基础规则

- 游戏共有 30 个结局。
- 四项国势为：兵力、粮草、民心、谋略。
- 四项数值低于等于 0 或高于等于 100 都会触发崩坏结局。
- 攻城失败会触发对应城池的失败结局。
- 部分隐藏结局需要多周目，死亡后获得的「兵法」会让后续周目更稳定。
- 结局判断有优先级：人物结局、歧途结局、隐藏结局通常会先于普通数值崩坏结算。

## 攻城要求

| 城池 | 结局相关 | 攻城要求 | 失败结局 | 胜利后 |
| --- | --- | --- | --- | --- |
| 枯骨关 | 第一城 | 兵力 ≥ 40 | 溃败 | 可遇见阿蛮 |
| 金沙渡 | 第二城 | 兵力 ≥ 50，粮草 ≥ 40 | 断粮 | 可遇见红袖 |
| 墨渊阁 | 第三城 | 谋略 ≥ 55，民心 ≥ 45 | 计中计 | 可遇见清越 |
| 天枢殿 | 第四城 | 民心 ≥ 55，谋略 ≥ 50 | 圣战 | 可遇见巫月 |
| 龙椅台 | 第五城 | 四项国势均 ≥ 55 | 功败垂成 | 进入统一类结局 |

选择“暂缓/等待”不会失败，但再次攻城会变难。

## 覆灭结局

| 结局 | 触发方式 |
| --- | --- |
| 兵变 | 兵力 ≤ 0 |
| 军阀 | 兵力 ≥ 100 |
| 饥荒 | 粮草 ≤ 0 |
| 腐粮 | 粮草 ≥ 100 |
| 暴动 | 民心 ≤ 0 |
| 圣人 | 民心 ≥ 100 |
| 中伏 | 谋略 ≤ 0 |
| 众叛 | 谋略 ≥ 100 |
| 溃败 | 攻打枯骨关失败 |
| 断粮 | 攻打金沙渡失败 |
| 计中计 | 攻打墨渊阁失败 |
| 圣战 | 攻打天枢殿失败 |
| 功败垂成 | 攻打龙椅台失败 |

想快速收集覆灭结局，可以在无兵法或少兵法的前几周目刻意偏科。数值波动较大，很容易冲破 0 或 100。

## 人物羁绊结局

人物需要先入局。四位人物分别在攻下前四座城后出现：

- 阿蛮：攻下枯骨关后，在「蛮族战俘」中选择留下。
- 红袖：攻下金沙渡后，在「商会千金」中选择留下。
- 清越：攻下墨渊阁后，在「才女」中选择留人。
- 巫月：攻下天枢殿后，在「祭司」中选择跟你走。

| 结局 | 关键条件 |
| --- | --- |
| 驯马 | 阿蛮好感 ≥ 75，兵力 ≥ 50，并在「缰绳」中选择臣服路线 |
| 反噬 | 阿蛮好感 ≥ 75，至少有 1 次暧昧/恋情计数，并在「缰绳」中选择沉沦路线 |
| 金屋 | 红袖好感 ≥ 75，粮草 ≥ 50，并在「金屋」中选择共掌天下账 |
| 债主 | 红袖好感 ≥ 75，至少有 1 次暧昧/恋情计数，并在「金屋」中选择欠她一生 |
| 折笔 | 清越好感 ≥ 75，谋略 ≥ 50，并在「折笔」中选择折笔入帐 |
| 间谍 | 清越好感 ≥ 75，至少有 1 次暧昧/恋情计数，并在「折笔」中选择写完那封信 |
| 破戒 | 巫月好感 ≥ 75，民心 ≥ 50，并在「破戒」中选择让她走下神坛 |
| 献祭 | 巫月好感 ≥ 75，至少有 1 次暧昧/恋情计数，并在「破戒」中选择带你上祭坛 |

人物线建议：想走臣服线，就尽量选“尊重但不沉溺”的选项；想走沉沦线，就多选带暧昧、亲近、共处意味的选项。

## 歧途结局

| 结局 | 触发方式 |
| --- | --- |
| 暴君 | 残酷值 ≥ 12，且兵力 ≥ 90 |
| 仁君 | 仁义值 ≥ 12，且民心 ≥ 90 |
| 酒池肉林 | 沉溺值 ≥ 10，且至少 3 位人物入局 |
| 空帐 | 至少 1 位人物入局，所有已入局人物好感都 < 30，且月份 ≥ 20 |

残酷值通常来自杀伐、强征、严刑、压制等选择。仁义值通常来自宽恕、赈济、保护百姓等选择。沉溺值通常来自人物亲密路线。

## 隐藏结局

| 结局 | 触发方式 |
| --- | --- |
| 一统天下 | 攻下五座城，且未满足更高优先级隐藏结局 |
| 割据一方 | 月份 ≥ 60，仍未统一天下，也没有先触发其它结局 |
| 千古一帝 | 第 3 周目后，攻下五城，月份 < 60，四项国势都在 30 到 70 之间 |
| 后宫之主 | 第 5 周目后，四位人物全部入局，且四人好感都 ≥ 60 |
| 弃甲归田 | 第 7 周目后，在「退路」中选择买下草鞋，获得退隐伏笔 |

注意：「弃甲归田」不是自动触发，必须亲自选择对应选项。

## 收集建议

- 前几局优先收集数值崩坏结局，顺便拿兵法。
- 想收攻城失败结局，就在对应城池要求不足时强行攻城。
- 想收人物结局，先保证能稳定攻下前四城，再集中培养目标人物好感。
- 想收「千古一帝」，不要把任一数值堆太高，攻下五城时四项都要保持在 30 到 70。
- 想收「后宫之主」，不要过早触发某一个人物专属结局，四人都入局后均衡刷好感。
- 想收「弃甲归田」，第 7 周目后留意「退路」，选择买下草鞋即可。
